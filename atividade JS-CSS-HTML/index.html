Atalhos do VSCode

Atalhos gerais.

Vamos ver alguns atalhos legais do VS Code que vão deixar a nossa vida mais alegre.

alt+z: Quebra automatica de linha.

alt+setas: Move a seleção cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha em que o cursor esta.

shift+alt+setas: Move uma copia da seleção cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha em que o cursor esta.

ctrl + espaço: chama o autocompletar de um contexto, como selecionar uma imagem ou um texto.

ctrl + ; : comentario / descomentario
